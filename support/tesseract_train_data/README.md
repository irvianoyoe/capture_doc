# Tesseract Train Data
* data train untuk nama(v.1.0) dan nik (v.1.5)

## Cara setting tesseract
* buka folder tesseract/tessdata
* copy file disini kesana

## requirement to run
* tesseract 3.x

## run on tesseract
* tesseract (input) stdout -l ind
* tesseract (input) stdout -l nama
